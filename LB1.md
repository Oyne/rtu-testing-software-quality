# Requirements for the final submission:
1. Create a Java class Calculator with basic arithmetic operations: addition, subtraction, multiplication and division (can be borrowed from the internet).
2. Create a test class and test one of the calculator's methods using JUnit for positive and negative partitions (where possible).
3. Add the Fibonacci number calculation method to the Calculator class (see code in slides).
4. Write all possible test cases for the Fibonacci method of the calculator using JUnit and AssertJ for assertions.
5. Add the Lucas number calculation method to the Calculator class.
6. Create the second test class and write all possible test cases for the Lucas method using TestNG.
7. Set priorities for TestNG methods and run the test class.
8. Set grouping by at least valid and invalid partitions and test different combinations of tests using TestNG.xml file.

# Evaluation during the project defence:
0-3: not ALL required tasks are done or explanations are unsatisfactory  
4-6: ALL required tasks are done and explanations are satisfactory, but not all test conditions are covered  
7-9: ALL required tasks are done, all test conditions are covered, and explanations are satisfactory  
10: ALL required tasks are done, all test conditions are covered, data providers & parameters are used in TestNG part, and explanations are satisfactory.
